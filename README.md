# NovaStack

NovaStack is a modern and responsive digital agency landing page built using Next.js and Tailwind CSS.

The website is designed for creators, startups, businesses, and personal brands who want meaningful digital experiences with a modern visual identity.
# Features

- Modern responsive design
- Dark premium UI with magenta theme
- Fully mobile responsive
- Smooth scrolling navigation
- Portfolio showcase section
- Testimonials section
- Contact form with simple validation
- Animated hover effects
- Clean reusable components
- Built using Next.js App Router

# Tech Stack
